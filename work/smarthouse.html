marthouse technologies (and everything that it involves) has always been a huge intererst of mine. As one of my last projects during my Master's program I worked along another PhD student in a smarthouse project with a distributed architecture. We used web technologies and iPOJO to do this.

My partner worked more on interfacing the iPOJO services with real-life components (house temperature, lighting, etc.). I worked mostly on the web platform and several iPOJO components that communicated with my partners services so that the user could manage all the services through the web.
